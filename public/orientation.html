VR Headset Preparation

Before entering the virtual museum, ensure the headset is connected and the system is ready.

  1. Session Code Verification: Ensure the 8-character code has been entered into the VR headset. (The code is saved in browser memory).
  2. Controls Practice: Practice voice commands (ASR), gaze control, and hand gestures in the lobby.
  3. Network Disconnect: Once in the lobby, the internet connection may be disconnected (Offline-Capable Operation).

When ready, you may start the VR session.
